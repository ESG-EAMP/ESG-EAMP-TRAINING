# Changelog

All notable changes to the PLATFORM PKSLestari  project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026

### Added

#### Assessment Features
- Group all category questions on a single page for improved user experience and navigation
- Admin can now view all previous assessment attempts for users who retake assessments
- Include assessment answers when downloading firm data to Excel
- Assessment Settings Page with custom question configuration:
  - Ability to disable other answers when "none of the above" type answer is present
  - Option to configure questions for single answer selection only
  - Option to configure questions for multiple answer selections
  - Version tracking and history for assessments

#### Admin Dashboard
- Display of total firm count and total assessment count in admin dashboard
- Password reset functionality in admin panel for user account management

#### User Management
- Account activation and deactivation capabilities in Manage User section
- Account verification functionality in Manage User section
- Firm profile viewing in Manage User section for verified accounts

### Changed

#### Admin Dashboard
- Admin sidebar: Changed "Learning center" to "Learning centre" (British spelling)
- Firms page: Clarified sorting functionality - sorting is by "total assessment" count, not individual assessment

### Fixed

#### Assessment Report
- Fixed blur and unclear display issue caused by animation in assessment reports
- Fixed footer and header misalignment when moving to next page (scaling issue resolved)

#### Admin Pages
- Fixed scrolling issue on `/admin/assessment/all-responses` page with new design implementation

#### UI/UX Improvements
- Fixed UI issues related to assessment history display
- Moved action columns to the first position in all tables for better usability and consistency
- Added loading state indicator to password reset button for better user feedback

### Removed

#### Admin Dashboard
- Removed "Achievements" from admin sidebar
- Removed achievement settings from admin panel
- Removed "latest assessment year" feature as per requirements
